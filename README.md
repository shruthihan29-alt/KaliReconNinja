## 📋 Project Overview

Kali Recon Ninja is a custom-built tool that automates the initial phases of penetration testing by chaining multiple powerful Kali Linux tools into a single professional workflow.

## ✨ Key Features

- **Automated Nmap** - Aggressive scanning with service, version, and OS detection
- **Gobuster** - Fast directory and file brute-forcing
- **Nikto** - Web server vulnerability scanning
- **Nuclei** - Modern template-based vulnerability scanner
- **WhatWeb** - Technology fingerprinting
- **EyeWitness** - Automated screenshot capture of discovered web pages
- **Professional HTML Report** - Clean, client-ready report generation
